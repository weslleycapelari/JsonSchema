# RADStudioJsonSchemaWizard

`RADStudioJsonSchemaWizard` is an IDE Extension Plugin (wizard) for RAD Studio (Delphi). It integrates JSON Schema validation, DTO generation, and schema analysis tools directly into the developer IDE.

## Features

- **Active File Validation**: Validate open JSON documents against schemas with a single click inside the IDE.
- **DTO Code Generation Shortcut**: Select a JSON Schema file in the project manager and instantly generate matching Delphi classes.
- **Draft Helper Wizard**: Provides assistance in writing schemas with auto-completion and schema structure validation.
- **Live Output Panel**: Directs validation issues and linter reports to the RAD Studio Messages view window.
