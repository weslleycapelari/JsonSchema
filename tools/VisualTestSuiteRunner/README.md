# VisualTestSuiteRunner

`VisualTestSuiteRunner` is an interactive GUI application (VCL/FireMonkey) written in Delphi. It runs the official JSON Schema Test Suite and displays a graphical dashboard of compliance status, keyword test cases, and errors.

## Features

- **Interactive Test Tree**: Browse tests by Draft version, file name, and specific keyword test cases in a graphical tree.
- **Detailed Diff Viewer**: View the exact input JSON, expected output, and the actual validation errors on test failures.
- **Performance Profiler**: Shows execution time for each draft's test runner, helping identify performance bottlenecks in validation routines.
- **Test Filtering**: Filter tests to show only failures, successes, or specific keywords under investigation.
