# Schema2Doc

`Schema2Doc` is a documentation generator that reads a JSON Schema and produces structured, interactive, and human-readable documentation in Markdown, HTML, or JSON formats.

## Features

- **Responsive HTML Generation**: Generates clean, searchable HTML pages documenting the structures.
- **Interactive Relationship Graphs**: Visualizes how schemas reference each other through `$ref` paths.
- **Constraint Tables**: Generates clean Markdown tables summarizing requirements (types, formats, constraints, default values).
- **Multi-lingual Support**: Supports generating documentation in English and Portuguese based on schema annotations.
